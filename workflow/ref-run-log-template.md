# Reference: Run Log Template

> Append this to the plan document in Step 6.

```markdown
## Run Log

### What went well (first-try wins, smooth flows)
- [what happened, why it worked]

### What went poorly (multi-attempt, user complaints, quality issues)
- [what happened, root cause of the struggle]

### Panelist Performance (if panel review ran)
| Panelist | Findings | Led to change | Rejected | Keep next time? |
|----------|----------|--------------|----------|-----------------|

### User Feedback
- [every correction, suggestion, opinion the user gave during the run]

### Bug Root Causes (if task was a bug fix)
- Bug: [description]
- Root cause: [why it existed]
- Prevention: [what rule/pattern would prevent this class of bug]

### Cost
- Total cost from `/cost`
```
